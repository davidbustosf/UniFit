# UniFit

## Aplicación móvil de fitness y rutinas de ejercicio

**Descripción breve:**  
UniFit ayuda a usuarios de todos los niveles a encontrar, seguir y registrar rutinas de ejercicio. La app se desarrollará en Android Studio con Firebase (Auth y Firestore) como backend.
